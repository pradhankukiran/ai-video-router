# Revideo project

Generated by ai-video-router. Claude Code is scoped to this directory and should iterate on the scenes under `src/scenes/`.

## Entry points

- `src/project.ts` — registers scenes.
- `src/scenes/example.tsx` — the default scene. Generator-based timeline (same API family as Motion Canvas).

## Conventions

- Each scene is a `function*` generator — `yield*` to drive time.
- Coordinate system is centered at (0, 0).
- Revideo is fork-compatible with Motion Canvas 2d components but adds server-rendering and an API-oriented CLI.

## Scripts

- `pnpm dev` → Vite dev server via `revideo serve`.
- `pnpm build` → headless render via `revideo render`.

## Assets

Drop images, audio, and fonts under `public/` and reference with `/asset.ext`.
