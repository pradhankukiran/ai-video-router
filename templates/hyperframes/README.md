# Hyperframes project

Generated by ai-video-router. Claude Code is scoped to this directory and should iterate on `index.html`.

## Entry points

- `index.html` — the composition. Everything renders from here.
- `hyperframes.config.json` — size, fps, duration, output path.

## Conventions

- The root element with `data-hf-scene` is the composition stage.
- Use `data-hf-from` / `data-hf-to` (seconds, relative to scene start) on any element to define a fade-in/out window.
- Use `data-hf-ease` for easing (`ease`, `ease-in`, `ease-out`, `linear`, …).
- Keep dimensions in sync across `hyperframes.config.json` and the `.stage` CSS.

## Scripts

- `pnpm dev` → `hyperframes preview` (live reload)
- `pnpm build` → `hyperframes render` (MP4 to `out/`)

## Assets

Drop images, audio, and fonts under `public/` and reference them with relative paths from `index.html`.
