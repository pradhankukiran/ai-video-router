# Remotion project

Generated by ai-video-router. Claude Code is scoped to this directory and should iterate by editing the files under `src/`.

## Entry points

- `src/index.ts` — registers the root. Do not change.
- `src/Root.tsx` — declares `<Composition id="Main" ... />`. Adjust `durationInFrames`, `fps`, `width`, `height`, and `defaultProps` here.
- `src/Composition.tsx` — the actual video. All visual changes go here.

## Conventions

- Use `useCurrentFrame()` + `interpolate()` for animation.
- Frame rate is 30 FPS by default (see `Root.tsx`).
- Keep the composition id as `"Main"` — the renderer picks it by id.

## Scripts

- `pnpm dev` → Remotion Studio at `http://localhost:3000` (note: router proxies this through its preview iframe).
- `pnpm build` → render MP4 to `out/` via the router.

## Assets

Drop images, audio, and fonts under `public/` and reference as `/asset.ext` from components.
