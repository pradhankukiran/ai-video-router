# Motion Canvas project

Generated by ai-video-router. Claude Code is scoped to this directory and should iterate on the scenes under `src/scenes/`.

## Entry points

- `src/project.ts` — registers scenes. Add new scenes here.
- `src/scenes/example.tsx` — the default scene. Generator-based timeline.

## Conventions

- Each scene is a `function*` generator — `yield*` to advance time with tween/await helpers.
- Coordinate system is centered at (0, 0).
- Prefer `createRef<T>()` + typed mutation (`ref().opacity(1, 0.5)`) over stateful JSX props.

## Scripts

- `pnpm dev` → Vite dev server with Motion Canvas UI.
- `pnpm build` → headless render to MP4.

## Assets

Drop images, audio, and fonts under `public/` and reference with `/asset.ext`.
