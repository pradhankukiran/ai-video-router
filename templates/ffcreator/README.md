# FFCreator project

Generated by ai-video-router. Claude Code is scoped to this directory and should iterate on `index.js`.

## Entry points

- `index.js` — builds the composition imperatively using `FFCreator`, `FFScene`, and child primitives (`FFText`, `FFImage`, `FFVideo`, etc.).

## Conventions

- `FFCreator` is the root. Add `FFScene`s to it; add children (text, image, video) to scenes.
- Coordinates are top-left origin. `width`/`height` configured at the creator level.
- Output path comes from `AVR_OUTPUT_PATH` env var when invoked by the router, or defaults to `out/video.mp4`.

## Scripts

- `pnpm build` → `node index.js`. Renders MP4 to the configured output path.

## Note

FFCreator has no live preview. Keep scene durations short while iterating, or render a single scene at a time.
