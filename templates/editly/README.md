# Editly project

Generated by ai-video-router. Claude Code is scoped to this directory and should iterate on `video.json5`.

## Entry points

- `video.json5` — declarative video spec. Editly renders this JSON5 config straight to MP4 via ffmpeg.

## Conventions

- Each entry in `clips[]` is one clip with an optional transition.
- Layers inside a clip compose vertically (video/image/title/subtitle/fill-color).
- See https://github.com/mifi/editly for the full schema reference.

## Scripts

- `pnpm build` → `editly video.json5 --out out/video.mp4`. The router's render pipeline invokes the same command.

## Note

Editly has no live preview — every iteration re-renders. For quick feedback, keep clips short while iterating.
